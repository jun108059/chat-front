## PnP (Yarn Berry)

참고 : https://toss.tech/article/node-modules-and-yarn-berry

### PnP 설정 켜기

```shell
$ npm install -g yarn
$ yarn set version berry
$ yarn install
```

```shell
# 실행
$ yarn start
```
